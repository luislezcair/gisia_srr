Open Patent Services (OPS)\n

\n\nOpen Patent Services (OPS) is a web service which provides access to the EPO's raw data via a standardised XML interface. It does this using RESTful architecture.\n\nRegister Log in Developers\u2019 testing area\n\nOPS data is extracted from the EPO's bibliographic, worldwide legal status, full-text and image databases. It is therefore from the same sources as the Espacenet and European Patent Register data.\n\n

With OPS you can:
